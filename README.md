# 🎓 College Event Management Portal - PMEC

A complete web-based Event Management Portal designed for **Parala Maharaja Engineering College (PMEC)**.  
This platform helps college admins manage events, and students can register, view events, and give feedback.

---

## 🌐 Live Preview

> 🔗 *To be added after deployment*  
Currently running locally at:  
`http://localhost/PMEC_Event_Management/index.php`

---

## 📌 Features

### 👩‍💼 Admin Features
- Login with email and password
- Add new events (title, date, description, club)
- View all student registrations
- View submitted feedback

### 👨‍🎓 Student Features
- View all upcoming events
- Register for events
- Submit feedback

### 📋 Public Pages
- Home Page (`index.php`)
- Event List
- Feedback form
- Registration form
